# Biofilm Poroelastic Simulation Code

Companion code for the manuscript:

**Biofilms as Living Sponges: A Drainage-Regime Framework for Mechanics, Transport and Intervention**

This repository contains the publication-final Python code and numerical outputs used to produce **Figure 2** and **Figure 3** of the manuscript. Figure 1 is conceptual and is not generated by this script.

## Overview

The code implements a minimal one-dimensional linearised Biot / Terzaghi consolidation model.
