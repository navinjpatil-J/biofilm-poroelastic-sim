# Biofilm Poroelastic Simulation Code

Companion code for the manuscript:

**Biofilms as Living Sponges: A Drainage-Regime Framework for Mechanics, Transport and Intervention**

This repository contains the publication-final Python code and numerical
outputs used to produce **Figure 2** and **Figure 3** of the manuscript.
Figure 1 is conceptual and is not generated by this script.

## Overview

The code implements a minimal one-dimensional linearised Biot / Terzaghi
consolidation model. It is used to illustrate:

- drainage-regime scaling (`tau_drain ~ L_eff^2 / D_p`),
- channel-spacing effects on interior drainage,
- the hydraulic-connectivity threshold (`Bi_sink = beta * s / D_p ~ 1`),
- experimental discriminants (tau ~ L^2 geometry test; tau ~ 1/k permeability test).

Validation is performed against the exact Terzaghi series for the
open-top / sealed-base case, with grid refinement for both hard and leaky
sinks.
